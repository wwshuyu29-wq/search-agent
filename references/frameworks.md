# 分析框架库

本文件是 SKILL.md 的引用资料，覆盖 **战略 / 营销 / 金融** 三大领域共 21 个分析框架。每个框架包含：适用场景、维度、搜索关键词模板、质量红线。运行时也可读取 `lib/frameworks.py` 获取核心 9 框架（Python dict）。

---

## 二、框架组合规则

复杂问题（命中"全面/系统/多维度"关键词，或 Top2 分数接近，或问题>50 字）优先推荐组合。

### 战略型组合

- **新市场进入组合** = PEST → Porter 五力 → SWOT → Ansoff
- **业务复盘组合** = BCG 波士顿矩阵 → GE 麦肯锡 → 价值链 → 7S
- **差异化战略组合** = 3C → 蓝海 ERRC → 4P → STP
- **全面战略组合** = PEST → Porter 五力 → 3C → SWOT

### 营销型组合

- **产品上市组合** = STP → JTBD → 4P → AIDA → Customer Journey
- **增长营销组合** = AARRR → RFM → Customer Journey → 4P
- **品牌资产诊断组合** = CBBE → STP → 4P → SWOT

### 金融型组合

- **投资判断组合** = 深度财报分析 → 杜邦分析 → 护城河 → DCF/相对估值 → 风险专项
- **风险扫描组合** = 财务比率四维 → Altman Z → Beneish M → 现金流质量 → 5C
- **深度财报组合** = 深度财报分析 → 3C 战略三角 → 杜邦分析

### 引导性扩框规则（意图单一时主动建议）

- 只问竞争格局 → 建议叠加 **SWOT**（补自身优劣势）
- 只问财报数字 → 建议叠加 **3C**（补竞争位置）+ **杜邦**（拆 ROE）
- 只问营销策略 → 建议叠加 **PEST**（补行业背景）+ **STP**（补客户细分）
- 只问用户增长 → 建议叠加 **JTBD**（补需求本质）+ **RFM**（补价值分层）
- 只问风险 → 建议叠加 **SWOT 决策矩阵** + **Altman Z-Score**
- 只问估值 → 建议叠加 **护城河**（判断可持续性）+ **杜邦**（拆盈利质量）

## 三、各框架详细定义

以下按 **战略 → 营销 → 金融** 顺序列出。核心 9 框架（★标记）与 `lib/frameworks.py` 内置配置对齐。

---

### ★ 1. 财报快评 [Sonnet]

- **视角**：投资者 / 财务分析师快速把脉
- **维度**：核心结论 / 三个关键发现 / 财务表现 / 增长驱动 / 利润变化 / 现金流 / 管理层指引 vs 实际 / 同行对比 / 主要风险 / 后续跟踪指标
- **搜索关键词模板**：
  - 财报：`{公司名} {期间} 财报`、`{公司名} {期间} earnings release`、`{TICKER} quarterly report`
  - 业绩会：`{公司名} {期间} earnings call transcript`、`{公司名} {期间} 业绩会`
- **质量红线**：财务数字必须标注**期间 + 币种 + 同比/环比口径**；未披露项目标"未披露"，不得推测。

### ★ 2. 深度财报分析 [Opus]

- **视角**：卖方分析师 / 尽调团队
- **维度**：核心结论 / 分析方法及信息来源 / 核心财务表现 / 历史对比 / 增长驱动 / 费用与利润质量 / 资产负债与现金安全垫 / 行业情况 / 同行对比 / 风险与反面证据 / 投资建议 / 后续跟踪指标
- **搜索关键词模板**：
  - 官方财报：`{公司名} {期间} annual report`、`{TICKER} 10-K`、`{公司名} 年报`
  - 业绩会：`{公司名} {期间} earnings call transcript`
  - 深度分析：`site:seekingalpha.com {TICKER} analysis`、`site:ft.com {公司名} earnings`
  - 同行：`{竞争对手} {期间} 财报`
  - 监管：`{公司名} 监管`、`{公司名} 公告`
- **思维链**：拆解 KPI 树 → 历史对比 → 同行对比 → 识别驱动因子 → 判断可持续性
- **质量红线**：结论必须区分**事实 / 计算 / 假设 / 判断**，四类分别标注。

### ★ 3. 同行竞争对比 [Opus]

- **视角**：战略咨询顾问
- **维度**：对比维度设计 / 核心指标对比表 / 相对优劣势 / 竞争格局 / 市场位置 / 核心竞争力 / 战略差异
- **搜索关键词模板**：
  - 公司 A：`{公司A} {期间} 财报`、`{公司A} 业务模式`
  - 公司 B：`{公司B} {期间} 财报`、`{公司B} 业务模式`
  - 行业对比：`{行业} 竞争格局`、`{行业} 市场份额`、`{公司A} vs {公司B}`
- **质量红线**：必须有具体数字对比；输出三栏对比表 + 关键差距分析 + 结论。

### ★ 4. PEST [Opus]

- **视角**：宏观策略研究员
- **维度**：政治 / 经济 / 社会 / 技术 / 综合影响 / 机会与威胁
- **搜索关键词模板**：
  - 政治：`{主题} 政策`、`{主题} 监管`、`{主题} 法规`
  - 经济：`{主题} 市场规模`、`{主题} 经济数据`、`{主题} 投资趋势`
  - 社会：`{主题} 用户需求`、`{主题} 消费习惯`
  - 技术：`{主题} 技术创新`、`{主题} 专利`、`{主题} 研发`
- **质量红线**：每维度 200-300 字，含 ≥1 条数据支撑；禁"可能影响 / 或许会"等模糊词。

### ★ 5. Porter 五力 [Opus]

- **视角**：行业分析师
- **维度**：供应商议价能力 / 买方议价能力 / 潜在进入者威胁 / 替代品威胁 / 行业内竞争 / 综合竞争强度
- **搜索关键词模板**：见"框架路由表"；补充 `{主题} 集中度 CR4`、`{主题} 议价能力`、`{主题} 进入壁垒`
- **质量红线**：每一力给出**高/中/低强度评估** + 至少 2 条依据。

### ★ 6. SWOT [Sonnet]

- **视角**：战略顾问
- **维度**：优势 / 劣势 / 机会 / 威胁 / 战略建议 / 优先级排序（可引入决策矩阵 impact × likelihood）
- **质量红线**：**每象限 ≥3 条具体条目**；输出 2×2 矩阵 + 战略启示（SO / WO / ST / WT 四条策略）。

### ★ 7. 3C 战略三角 [Opus]

- **视角**：大前研一式战略咨询
- **维度**：公司 / 客户 / 竞争对手 / 三方匹配度 / 战略卡位
- **质量红线**：三方分析必须**闭环**——"公司能力 vs 客户需求 vs 竞对差距"三者要有明确 mapping。

### ★ 8. 4P 营销 [Sonnet]

- **视角**：品牌经理 / GTM 负责人
- **维度**：产品 / 价格 / 渠道 / 推广 / 营销组合评估 / 优化建议
- **质量红线**：4P 之间要有一致性论证（如高端定价必须匹配高端渠道 + 品牌推广）。

### ★ 9. 风险专项 [Opus]

- **视角**：风控/合规负责人
- **维度**：监管 / 财务 / 经营 / 市场 / 技术 / 风险量化评估 / 应对建议
- **质量红线**：输出**决策矩阵**（impact × likelihood 排序），每条风险标注**触发条件 + 预警指标**。

---

### 10. Ansoff 增长矩阵 [Sonnet]

- **视角**：CEO / 战略部
- **维度**：市场渗透（现有市场 + 现有产品）/ 市场开发（新市场 + 现有产品）/ 产品开发（现有市场 + 新产品）/ 多元化（新市场 + 新产品）
- **搜索关键词模板**：`{公司} 新市场`、`{公司} 新产品`、`{公司} 收购`、`{公司} 出海`、`{公司} 品类扩张`
- **质量红线**：四象限每项给出**风险等级**（多元化最高）+ 资源需求评估。

### 11. BCG 波士顿矩阵 [Sonnet]

- **视角**：多业务集团投资决策
- **维度**：明星（高增长+高份额）/ 金牛（低增长+高份额）/ 问号（高增长+低份额）/ 瘦狗（低增长+低份额）
- **搜索关键词模板**：`{公司} 业务分部`、`{公司} 收入构成`、`{行业} 增长率`、`{业务} 市场份额`
- **质量红线**：必须画出 2×2 图或表格，每个业务标注**营收占比 + 增长率 + 市占率**。

### 12. GE 麦肯锡矩阵 [Opus]

- **视角**：多业务组合精细化管理（BCG 增强版）
- **维度**：行业吸引力（市场规模/增长率/盈利性/竞争强度）× 业务竞争力（份额/品牌/成本/技术），3×3 九宫格
- **质量红线**：吸引力和竞争力必须**加权评分**（各因子权重合计 100%）；输出投资 / 选择性投资 / 收割/退出 三类决策。

### 13. 麦肯锡 7S [Opus]

- **视角**：组织诊断 / 变革管理
- **维度**：Strategy 战略 / Structure 结构 / Systems 系统 / Shared Values 共同价值观 / Style 风格 / Staff 员工 / Skills 技能
- **搜索关键词模板**：`{公司} 组织架构`、`{公司} 企业文化`、`{公司} 高管`、`{公司} 人才战略`、`{公司} 流程`
- **质量红线**：7 个 S 之间必须做**一致性分析**——软要素（Shared Values / Style / Staff / Skills）与硬要素（Strategy / Structure / Systems）是否对齐。

### 14. Porter 价值链 [Opus]

- **视角**：识别价值创造与成本源头
- **维度**：主活动（内部物流·运营·外部物流·营销销售·服务）+ 支持活动（企业基础设施·人力资源·技术开发·采购）
- **质量红线**：每一环节必须标注**成本占比或价值增量**，识别 2-3 个核心差异化环节。

### 15. 蓝海战略 ERRC [Opus]

- **视角**：品类创新 / 差异化
- **维度**：Eliminate 剔除 / Reduce 减少 / Raise 增加 / Create 创造（相对行业标准的四步动作）
- **质量红线**：必须先做"战略画布"——列出行业竞争因子，标出行业平均水平 vs 目标产品水平；ERRC 四类动作各 ≥1 条。

---

### 16. STP 市场细分定位 [Opus]

- **视角**：品牌战略 / 产品营销
- **维度**：Segmentation 细分（人口/地理/心理/行为四类变量）→ Targeting 目标市场选择（吸引力 × 匹配度）→ Positioning 定位（差异化价值主张 + 感知地图）
- **搜索关键词模板**：`{品牌} 目标用户`、`{品牌} 用户画像`、`{品牌} 定位`、`{行业} 细分市场`、`{品牌} 感知地图`
- **质量红线**：细分必须**互斥且穷尽**（MECE）；目标市场至少给出 3 个吸引力指标评分；定位陈述遵循 "对 [目标客户]，[品牌] 是 [品类] 中 [差异点]" 句式。

### 17. AIDA 广告漏斗 [Sonnet]

- **视角**：广告主 / 增长团队
- **维度**：Attention 注意 / Interest 兴趣 / Desire 渴望 / Action 行动（现代变体：AIDAS 加满意 / AISAS 加搜索共享）
- **搜索关键词模板**：`{品牌} 广告效果`、`{品牌} 曝光量`、`{品牌} 点击率`、`{品牌} 转化率`
- **质量红线**：每一层必须给出**核心指标**（Attention=曝光/CPM，Interest=CTR，Desire=停留/加购，Action=转化率/CPA）+ 行业基准对比。

### 18. AARRR 海盗指标 [Opus]

- **视角**：增长负责人 / 产品经理
- **维度**：Acquisition 获取 / Activation 激活 / Retention 留存 / Revenue 收入 / Referral 传播
- **搜索关键词模板**：`{产品} DAU/MAU`、`{产品} 留存率`、`{产品} 获客成本 CAC`、`{产品} LTV`、`{产品} NPS`、`{产品} 病毒系数 K`
- **质量红线**：每环节必须给出**关键指标 + 漏斗转化率**；LTV/CAC 比值 > 3 视为健康；识别漏斗最薄弱环节。

### 19. RFM 客户价值分层 [Sonnet]

- **视角**：CRM / 精准营销
- **维度**：Recency 最近购买时间 / Frequency 购买频次 / Monetary 消费金额（每维度 5 分位，共 125 类，可简化为 8 类：重要价值/重要保持/重要发展/重要挽留/一般价值/一般保持/一般发展/一般挽留）
- **搜索关键词模板**：`{品牌} 会员体系`、`{品牌} 复购率`、`{品牌} 客单价`、`{品牌} VIP 用户`
- **质量红线**：三维度打分方法必须明确（分位数 or 阈值）；每类客户给出**差异化营销策略**。

### 20. JTBD 待完成的任务 [Opus]

- **视角**：产品经理 / 用户研究
- **维度**：功能性任务 / 情感性任务 / 社会性任务 / 情境（时间/地点/触发条件）/ 痛点 / 收获 / 竞争性方案
- **搜索关键词模板**：`{产品} 用户评论`、`{产品} 使用场景`、`{产品} 用户访谈`、`{产品} 替代方案`
- **质量红线**：任务陈述必须遵循 "当 [情境] 时，我想 [动机]，以便 [预期结果]" 句式；识别当前解决方案的"雇佣与解雇"事件。

### 21. Keller CBBE 品牌金字塔 [Opus]

- **视角**：品牌资产管理
- **维度**：底层 Salience 显著性 → 中层 Performance 绩效 / Imagery 形象 → 上层 Judgments 评判 / Feelings 感受 → 顶层 Resonance 共鸣（4 层 6 模块）
- **搜索关键词模板**：`{品牌} 认知度`、`{品牌} 品质`、`{品牌} 形象`、`{品牌} NPS`、`{品牌} 用户忠诚度`
- **质量红线**：每一层必须有**量化指标或访谈证据**支撑；识别品牌当前所在层级 + 向上突破的瓶颈。

### 22. Customer Journey Map 用户旅程 [Sonnet]

- **视角**：CX 设计 / 全渠道运营
- **维度**：阶段（认知·考虑·购买·使用·忠诚）× 触点 × 用户行为 × 用户情绪 × 痛点 × 机会点
- **质量红线**：必须画出旅程图（表格或图形）；每阶段至少标注 1 个**痛点** + 1 个**机会点**。

---

### 23. DuPont 杜邦分析 [Opus]

- **视角**：ROE 结构分解
- **维度**：ROE = **净利率（盈利能力）× 资产周转率（运营效率）× 权益乘数（财务杠杆）**；进阶五步法：ROE = 税负 × 利息负担 × EBIT 利润率 × 资产周转 × 权益乘数
- **搜索关键词模板**：`{公司} ROE`、`{公司} 净利率`、`{公司} 总资产周转率`、`{公司} 资产负债率`
- **质量红线**：必须列出近 3-5 年趋势 + 同行对比；识别 ROE 是**盈利驱动 / 效率驱动 / 杠杆驱动** 中的哪一类。

### 24. 财务比率四维 [Sonnet]

- **视角**：财务健康度全景扫描
- **维度**：
  - **盈利能力**：毛利率、净利率、ROE、ROA
  - **偿债能力**：流动比率、速动比率、资产负债率、利息保障倍数
  - **营运能力**：总资产周转率、应收账款周转率、存货周转率
  - **成长能力**：营收增长率、净利润增长率、总资产增长率
- **质量红线**：每比率必须给出**期间 + 计算公式 + 行业均值对比 + 趋势判断**。

### 25. DCF 现金流折现估值 [Opus]

- **视角**：内在价值投资
- **维度**：显性预测期（5-10 年自由现金流 FCF）+ 终值（Gordon 增长模型 or 退出倍数）+ WACC 折现率 + 敏感性分析（增长率 × WACC 网格）
- **搜索关键词模板**：`{公司} FCF 自由现金流`、`{公司} WACC`、`{公司} 永续增长率`、`{行业} risk premium`
- **质量红线**：必须给出**基准 / 乐观 / 悲观 三情景估值**；WACC 各参数（Rf、β、市场风险溢价）分别标注来源。

### 26. 相对估值倍数 [Sonnet]

- **视角**：跨公司横向估值
- **维度**：P/E（盈利）、P/B（净资产）、EV/EBITDA（企业价值）、PEG（成长调整）、P/S（早期公司）+ 同行分位数比较
- **搜索关键词模板**：`{公司} P/E`、`{行业} P/E 中位数`、`{公司} 估值`、`{公司} valuation multiples`
- **质量红线**：必须**同行同期**对比（至少 5 家可比公司）；说明为何该行业适用某个倍数（如亏损期用 P/S、周期股用 P/B）。

### 27. 巴菲特护城河 [Opus]

- **视角**：长期投资 / 竞争壁垒
- **维度**：无形资产（品牌/专利/牌照）/ 成本优势 / 网络效应 / 切换成本 / 有效规模
- **搜索关键词模板**：`{公司} 品牌溢价`、`{公司} 专利`、`{公司} 网络效应`、`{公司} 用户粘性`、`{公司} 规模经济`
- **质量红线**：识别到的护城河必须给出**数据证据**（如毛利率长期高于同行、切换成本用户调研）；判断**护城河是加宽还是变窄**。

### 28. 现金流质量分析 [Opus]

- **视角**：识别财务粉饰
- **维度**：经营性现金流 / 净利润 匹配度（>1 视为质量高）、应收账款周转率变化、存货周转率变化、经营性现金流覆盖资本开支率、自由现金流稳定性
- **搜索关键词模板**：`{公司} 经营性现金流`、`{公司} 应收账款`、`{公司} 存货`、`{公司} 现金流风险`
- **质量红线**：连续 3 年经营性 CF / 净利润 < 0.8 → 标注**红旗警告**；应收账款增速 > 营收增速 2x → 收入质量存疑。

### 29. Beneish M-Score + Piotroski F-Score [Opus]

- **视角**：舞弊筛查 & 价值股筛选
- **维度**：
  - **Beneish M-Score**（8 指标）：DSRI 应收账款周转、GMI 毛利率恶化、AQI 资产质量、SGI 收入增长、DEPI 折旧变化、SGAI 销管费用、LVGI 杠杆、TATA 应计项目。M > -1.78 → 造假嫌疑。
  - **Piotroski F-Score**（9 项二元指标）：盈利 4 项 + 杠杆流动性 3 项 + 运营效率 2 项，得分 8-9 为优质，0-2 需回避。
- **质量红线**：必须列出**每一项指标计算结果 + 得分**；结论标注置信度（数据源可靠性）。

### 30. Altman Z-Score 破产预警 [Sonnet]

- **视角**：违约/破产概率评估
- **维度**：Z = 1.2A + 1.4B + 3.3C + 0.6D + 1.0E，其中 A=营运资本/总资产、B=留存收益/总资产、C=EBIT/总资产、D=股权市值/总负债、E=销售/总资产。Z > 2.99 安全，1.81-2.99 灰色区，< 1.81 高风险。
- **搜索关键词模板**：`{公司} 财务困境`、`{公司} 债务违约`、`{公司} 资产负债表`
- **质量红线**：适用**上市制造企业**；非制造/新兴市场需用 Z' 或 Z'' 变体，必须说明使用的模型版本。

### 31. 5C 信贷分析 [Sonnet]

- **视角**：银行信贷 / 供应商授信 / 合作评估
- **维度**：Character 品格（管理层诚信/信用记录）/ Capacity 偿债能力（现金流/覆盖倍数）/ Capital 资本（自有资金/杠杆）/ Collateral 抵押品 / Conditions 外部环境（行业周期/宏观）
- **搜索关键词模板**：`{公司} 信用评级`、`{公司} 违约记录`、`{公司} 管理层背景`、`{公司} 担保`、`{行业} 周期`
- **质量红线**：每 C 给出**证据 + 打分**（如 1-5 分），最终综合判断"授信/拒绝/附条件"。

## 五、扩展框架（后续路线图）

- **麦肯锡 MECE 原则**：任何拆解必须相互独立、完全穷尽

- **OKR / KPI 树对齐**：目标层层拆解
- **PESTLE**：PEST + 法律 Legal + 环境 Environmental
- **VRIO 资源基础观**：Value/Rarity/Imitability/Organization
- **CAMELS 银行评级**：Capital / Asset / Management / Earnings / Liquidity / Sensitivity（银行专用）
